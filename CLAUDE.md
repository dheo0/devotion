# 프로젝트: DEVOTION

하루 성경말씀을 읽고 그 내용을 요약 및 실천사항을 적는 메모 성격의 앱입니다.

## 코드 스타일

- TypeScript strict 모드 사용, `any` 타입 금지
- default export 대신 named export 사용
- CSS: Tailwind 유틸리티 클래스 사용, 커스텀 CSS 파일 금지
